# Simulación de lanzamiento de martillo

Se simula la dinámica de un martillo olímpico esférico lanzado oblicuamente, resultando en un movimiento parabólico. Se consideran tres regímenes para el coeficiente de rozamiento del medio: nula (en el vacío), con flujo laminar y flujo inestable oscilante.
Se utiliza el método de Euler para la resolución del sistema de ecuaciones diferenciales que modelan la dinámica del objeto.
