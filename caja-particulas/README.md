# Partículas en una caja

Simulación estocástica del transporte de partículas entre dos regiones de una caja dividida por una pared con un pequeño agujero. El sistema se modela como una cadena de Markov: en cada paso de tiempo, una partícula pasa de un lado a otro según una probabilidad que depende del número de partículas en cada lado.

Se realizaron 10 simulaciones y se comparó el promedio contra la solución analítica del sistema en equilibrio.
