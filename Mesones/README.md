# Supervivencia de piones y prueba de aleatoriedad

Simulación numérica por medio de variables pseudoaleatorias para determinar el número de piones que sobreviven a cierto desplazamiento bajo dos distribuciones distintas de energía: uniforme y gaussiana.
Se determina que un conjunto de números no es aleatorio.

## Archivos
- `Instrucciones tarea 7.pdf` — enunciado del problema
- `Pion_a.py` — implementación del código para partículas monoenergéticas
- `Pion_b.py` — implementación del código para energías con distribución gaussiana
- `Prueba.py` — implementación del esquema numérico
- `Mesones y prueba.pdf` — desarrollo analítico y resultados
