# Evolución de temperatura en una varilla

Simulación numérica de la evolución de temperatura a lo largo de una varilla mediante la ecuación de difusión (ecuación de calor), resuelta con un esquema de diferencias finitas.

## Archivos
- `difcalnum.py` — implementación del esquema numérico
- `EcalAn.py`- solución analítica graficada
- `Difusión de calor.pdf` — desarrollo analítico
