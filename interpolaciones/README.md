# Interpolaciones bajo distintos métodos

Se realizan ajustes de interpolación por medio del polinomio interpolante de Lagrange, splines cúbicos y mínimos cuadrados no lineales ponderados para la fórmula de resonancia de Breit-Wigner.

## Archivos
- `Interpolaciones.pdf`- instrucciones del problema (ver sólo los incisos 1, 2 y 3)
-  `BretWigner.txt`- Archivo de texto con los datos a leer para realizar el ajuste
- `Lagranja.py`- interpolación por el polinomio de Lagrange
- `CubicSplines.py`- interpolación por splines cúbicos
- `NewtonR.py`- solución por medio de mínimos cuadrados y el método de Newton Raphson multidimensional para la optimización de parámetros
- `interp_5.pdf`- desarrollo analítico a mano
